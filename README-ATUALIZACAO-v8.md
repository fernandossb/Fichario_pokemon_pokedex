# Versão 8 — Preços multifonte

- Liga Pokémon deixou de ser ponto único de falha.
- Cardmarket e TCGplayer são consultados gratuitamente via TCGdex.
- Conversão EUR/BRL e USD/BRL com cache diário.
- Quando Cardmarket e TCGplayer respondem, o app usa a média convertida.
- Se a Liga responder, ela continua sendo priorizada por representar o mercado brasileiro.
- O último preço salvo permanece disponível quando uma consulta falha.
- Pokédex e Decks foram preservados.
