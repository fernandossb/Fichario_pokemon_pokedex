# Versão 6

- Pokédex preservada e validada no build.
- Parser de preços da Liga Pokémon mais tolerante a mudanças de texto e quantidade de valores.
- Rotas alternativas de consulta e mensagens de diagnóstico detalhadas.
- Aba Decks funcional: gera deck a partir da coleção, edita quantidades, adiciona/remove cartas e valida total de 60.
- O gerador usa somente cartas cadastradas e nunca excede a quantidade possuída.

Observação: como a Liga Pokémon não oferece API pública estável para esta consulta, mudanças ou bloqueios no site ainda podem exigir novos ajustes.
