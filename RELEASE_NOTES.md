# Fichário Pokémon v1.1.0

- Integração com o Pokémon Price Database Brasil.
- Preço Brasil como fonte principal por ID exato e acabamento.
- Cache offline da tabela central em IndexedDB.
- Sincronização automática quando o banco recebe uma atualização.
- Cardmarket e TCGplayer mantidos como fallback.
- Painel com data, cobertura e pendências do banco de preços.
