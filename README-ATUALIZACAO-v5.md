# Fichário Pokémon v5

Esta versão combina a base instalável validada no Poco X7/Android 16 com o checkpoint mais recente.

## Requisitos preservados
- Aba Pokédex e 1.025 sprites.
- Cadastro detalhado por variante/acabamento.
- Fotos e imagens das cartas.
- Atualização online do catálogo.
- Consulta de preços brasileiros pela Liga Pokémon, sem usar a Liga para identificar a carta.
- Backup e restauração manual.

## Privacidade
- O APK não contém a coleção pessoal do desenvolvedor/usuário.
- Instalações novas começam com coleção vazia.
- Backup automático do Android foi desativado para evitar restauração inesperada em outro telefone.

## Build
O GitHub Actions valida os requisitos e gera um APK debug assinado automaticamente.
