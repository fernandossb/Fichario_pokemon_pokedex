# Versão 7 — preços multifonte

## Fontes gratuitas

- Liga Pokémon (Brasil), quando disponível.
- Cardmarket via TCGdex.
- TCGplayer via TCGdex.
- Câmbio EUR/BRL e USD/BRL com cache diário.

A falha da Liga Pokémon não interrompe mais a atualização. O aplicativo consulta as fontes em paralelo, conserva o último preço salvo e mostra a origem usada.

## Decks

O editor e o gerador de decks da versão 6 foram preservados.

## Segurança

A Pokédex com 1.025 sprites e a coleção local foram preservadas. Nenhum banco pessoal é distribuído no APK.
