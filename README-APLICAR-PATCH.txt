ATUALIZADOR INTERNO - COMO APLICAR

1. Copie todo o conteúdo deste patch para a pasta do repositório.
2. Confirme a substituição dos arquivos.
3. No GitHub Desktop: Commit to main e Push origin.
4. O Actions criará o APK assinado e uma Release oficial do GitHub.
5. Esta versão ainda precisa ser instalada manualmente uma vez.
6. A partir da próxima versão, o próprio aplicativo avisará e baixará a atualização.

IMPORTANTE: em toda versão futura aumente o versionCode em app/build.gradle e atualize RELEASE_NOTES.md.
