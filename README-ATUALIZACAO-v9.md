# Versão 9 — Deck Builder

Esta versão mantém a Pokédex e o motor de preços multifonte da versão 8 e melhora a aba Decks.

## Novidades
- Geração de deck com foco opcional por tipo de Pokémon.
- Usa somente cartas cadastradas no fichário.
- Limite de 60 cartas.
- Limite de 4 cópias pelo mesmo nome, somando artes/coleções diferentes.
- Validação de quantidade possuída.
- Relatório de erros e avisos.
- Nota de força/consistência de 0 a 100.
- Editor com adicionar/remover cartas.
- Renomear, duplicar e exportar lista do deck.
- GitHub Actions valida Pokédex, preços e Deck Builder antes de compilar.

## Observação
O catálogo atual possui nome, coleção, número, imagem e vínculo com Pokémon. Como ainda não contém ataques, habilidades, estágio e custo de energia de todas as cartas, a nota é uma estimativa baseada em composição, tipo, quantidade e nomes de cartas de suporte. Uma versão futura poderá enriquecer os dados via TCGdex para análises competitivas mais profundas.
